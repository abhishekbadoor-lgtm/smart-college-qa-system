# 🎓 Smart College Information & Q&A System

A Django-based web application designed to streamline college management and enhance student–faculty interaction through an integrated Q&A platform.

---

## 🚀 Features

* 📊 Student Dashboard (attendance, marks, notices)
* 👨‍🏫 Faculty Dashboard (manage students, attendance, marks, notices)
* 💬 Q&A Forum for student queries
* 🧑‍💼 Custom Admin Panel
* 📝 Feedback system

---

## 🛠️ Tech Stack

* Backend: Django (Python)
* Frontend: HTML, CSS
* Database: SQLite

---

## ⚙️ How to Run the Project

```bash
git clone https://github.com/abhishekbadoor-lgtm/smart-college-qa-system.git
cd smart-college-qa-system
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
